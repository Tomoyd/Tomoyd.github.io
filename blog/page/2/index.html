encodeURI和encodeComponentURI不同

May 27, 2020 · 1 min read

huy

忆昔当年长沙镇，算来不觉有数春

encodeURIComponent()#

除了A-Z a-z 0-9 - _ . () ! ~ ' *可以编码任意字符串

Live Editor
function TestEncode() {
  return <div>{encodeURIComponent("=")}</div>;
}
Result
%3D

encodeURI()#

除了A-Z a-z 0-9 - _ . () ! ~ ' * = + : ? @ # & ; , : $ /，可以编码任意字符串

Live Editor
function TestEncode() {
  return <div>{encodeURI("=")}</div>;
}
Result
