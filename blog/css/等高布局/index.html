等高布局几种方案

May 29, 2020 · 1 min read

huy

忆昔当年长沙镇，算来不觉有数春

利用 padding-bottom margin-bottom#

将项目的 padding-bottom 设置为一个足够大的值,让较低的元素足够高，再利用 margin-bottom 将其布局拉回

Live Editor
<>
<style>
    {`
      .sec1{
        overflow:hidden;
      }
      .sec1 div{
          color:red;
          background:blue;
          display:inline-block;
        }
        .item1{
          padding-bottom:99999px;
          margin-bottom:-99999px;
        }
        .sec1 .item2{
          background:yellow;
          height:40px
        }
    `}
</style>
<section className="sec1">
  <div className="item1">123</div>
  <div className="item2">88</div>
</section>
</>
Result
123
88

display: grid 布局方式#

设置父元素为 display:grid 在设置按照文档流列排布

Live Editor
function DisplayGrid() {
  const gridStyle = {
    display: "grid",
    gridAutoFlow: "column",
    gridTemplateColumns: "100px auto"
  };
  return (
    <div style={gridStyle}>
      <div style={{ color: "#ff0000" }}>3</div>
      <div style={{ color: "#ffff00" }}>11111111</div>
    </div>
  );
}
Result
3
11111111

display: table 子元素指定为 display:table-cell#

Live Editor
<>
<style>
    {`
      .table{
        display:table;
      }
      .table div{
         width:100px;
         background:yellow;
         display:table-cell
        }
      .table .tableItem2{
        background:red;
        height:40px;
      }
    `}
</style>
<section className="table">
  <div >123</div>
  <div className="tableItem2">88</div>
</section>
</>
Result
123
88
